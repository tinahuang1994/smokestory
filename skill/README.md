# SmokeStory Skill

This folder contains the reusable skill documentation extracted from building SmokeStory.

## Files

SKILL.md — The main skill document.
Covers the full three-layer architecture: data pipeline, narrative engine, and financial impact module. Includes deployment lessons, testing patterns, and science communication rules.

## Who This Is For

Developers who want to build a similar climate data intelligence tool for a different hazard type or region.

Journalists who want to understand how SmokeStory works and evaluate its methodology.

Climate researchers who want to adapt the financial impact methodology for their own analysis. Note: SmokeStory was only validated for wildfire smoke in California. Adapting it for other hazards requires identifying appropriate data sources and validating outputs for that specific domain.
